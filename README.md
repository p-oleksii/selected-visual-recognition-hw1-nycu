## Introduction
The main task for this assignment was image classification using ResNet as the model backbone. There are three datasets provided: training, validation, and testing. After successfully training and validating the model on the corresponding datasets, we run the model on the test dataset and output all predictions into a CSV file, which is then submitted to the competition system.

My model uses a standard ResNet-50 with a custom classifier and fine-tuned training parameters. The main idea was to leverage the model’s lightweight architecture while enhancing it with a custom classifier to reduce overfitting and balance overall performance with model size.
